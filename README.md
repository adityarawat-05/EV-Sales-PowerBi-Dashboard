# ⚡ Electric Vehicle Sales Analysis Dashboard

A data analytics project that explores **Electric Vehicle (EV) sales trends** using interactive dashboards built in Microsoft Power BI.

This project analyzes EV sales across different **states, vehicle categories, and time periods** to identify adoption patterns and provide meaningful insights through data visualization.

---

## 📊 Project Overview

Electric Vehicles are rapidly transforming the transportation industry as countries move toward **sustainable and eco-friendly mobility solutions**. However, analyzing large volumes of EV sales data manually can be difficult.

This project converts raw EV sales data into an **interactive Power BI dashboard** that helps users easily understand:

- EV adoption trends over time
- State-wise EV sales distribution
- Popular EV vehicle categories
- Monthly and yearly sales patterns

The dashboard allows users to explore the data dynamically using filters and slicers.

- Microsoft Power BI

---

## 📂 Dataset Information

The dataset contains EV sales information with the following attributes:

- Year
- Month Name
- Date
- State
- Vehicle Class
- Vehicle Category
- EV Sales Quantity

The data is cleaned and processed before visualization to ensure accurate insights.

---

## 📈 Dashboard Features

The Power BI dashboard includes several visualizations such as:

- 📉 **EV Sales Trend Analysis**
- 📊 **Top 10 States by EV Sales**
- 🥧 **Vehicle Category Distribution**
- 🌍 **State-wise EV Adoption**
- 📅 **Monthly Sales Analysis**

Interactive slicers allow users to filter data by:

- Year
- State
- Vehicle Category
- Vehicle Class

---

## 📊 Key Insights

Some important insights derived from the dashboard include:

- EV adoption has increased steadily over recent years.
- Certain states contribute significantly to overall EV sales.
- Two-wheelers dominate EV adoption compared to other vehicle categories.
- Monthly sales patterns highlight seasonal variations in EV demand.
